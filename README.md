# Large-Scale Audit Document Processing System

## Overview

Developed a scalable audit document processing system for handling and validating **80,000+ audit documents** using automated parsing, filtering, and classification workflows.
The system streamlined Inspection Report (IR) segregation, metadata validation, and structured data extraction from large-scale unstructured document datasets.



## Features

* Automated processing of 80,000+ audit documents
* Inspection Report (IR) segregation and filtering
* Regex-based document parsing and classification
* Metadata validation and cross-verification
* Structured data extraction from unstructured documents
* High-volume document preprocessing workflows
* Scalable pipeline for enterprise-scale audit processing



## Tech Stack

* Python
* Regex
* Streamlit



## Workflow

1. Large-scale document ingestion
2. Document preprocessing and cleaning
3. Regex-based content extraction
4. Inspection Report (IR) classification
5. Metadata validation and consistency checks
6. Structured data generation
7. Cross-verification and final output
